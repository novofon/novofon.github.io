    Список доступных методов

    Звонки

    Метод Описание
    "upload.calls" Загрузка звонков и разговоров для сквозной и речевой аналитики
    "get.uploaded_calls" Получение информации о загрузке звонков
    "get.calls_report" Получение отчёта по сессиям звонков "Список обращений" -> "Звонки"
    "create.call_session_recognition_task" Создание задачи на распознавание сеанса звонка

    Офлайн-заявки

    Метод Описание
    "upload.offline_messages" Загрузка офлайн-заявок
    "delete.offline_messages" Удаление офлайн-заявки
    "get.offline_messages_report" Получение информации по оффлайн заявкам

    Кастомные поля у Офлайн-заявок

    Метод Описание
    "create.offline_message_user_field" Создание кастомного поля
    "delete.offline_message_user_field" Удаление кастомного поля
    "update.offline_message_user_field" Обновление кастомного поля
    "get.offline_message_user_fields" Получение информации по кастомным полям оффлайн заявок

    Чаты

    Метод Описание
    "upload.chats" Загрузка чатов
    "delete.chats" Удаление чатa
    "get.chats_report" Получение информации о чатах
    "get.chat_messages_report" Получение всех сообщений чата
