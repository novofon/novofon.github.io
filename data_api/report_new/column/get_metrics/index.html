    Получение списка метрик

    Получение списка метрик > Получение списка метрик
    Метод get.metrics
    Описание Получение списка метрик
    Кому доступен Агент, Клиент
    Необходимые компоненты для работы с методом Data API для настройки
    Вес метода 1 балл

    Параметры запроса

    Название Тип Обязательный Описание
    access_token string да Ключ сессии аутентификации
    user_id number нет Уникальный идентификатор пользователя клиента агента от имени которого делается запрос
    Является обязательным для агента
    Для получения списка пользователей клиента необходимо использовать метод "get.customer_users"
    report_type enum да Тип отчета пользователя. Для получения списка отчетов пользователей необходимо использовать метод "get.reports_list"

    Параметры ответа

    Название Тип Допустимые значения Ответ по умолчанию Описание
    id enum да Уникальная мнемоника метрики
    name string да Название метрики
    data_type enum да Формат данных
    Список агрегатов доступный для метрики
    aggregates array да Список агрегатов доступный для выбранной метрики
    id enum Уникальная мнемоника агрегата
    name string Название агрегата
    has_percentage boolean Возможен ли расчет доли
    is_default boolean Агрегат по умолчанию
    Список способов мультиканального анализа доступных для метрики
    multichannel_models object да Список способов мультиканального анализа доступных для метрики.
    Способы мультиканального анализа
    models array да Способы мультиканального анализа.
    id enum Уникальная мнемоника мультиканального анализа
    name string Название мультиканального анализа
    description string Описание мультиканального анализа
    is_default boolean Мультиканальный анализ по умолчанию
    group_id string Группа мультиканального анализа
    sort number Сквозной идентификатор, используемый для сортировки в дереве.
    Группы мультиканального анализа
    models_groups array да Список групп мультиканального анализа
    id string Мнемоника группы мультиканального анализа
    name string Название группы мультиканального анализа
    description string Описание группы мультиканального анализа.
    parent_group_id number Идентификатор группы во вложении которой находится данная группа мультиканального анализа
    sort number Сквозной идентификатор, используемый для сортировки.

    Список возможных агрегатов

    Мнемоника Название Используется ли расчет доли Описание
    count Количество Да
    distinctcount Количество уникальных Да
    sum Сумма Да
    avg Среднее
    median Медиана
    min Минимум
    max Максимум

    JSON структура запроса

    {
    "jsonrpc":"2.0",
    "id":"number",
    "method":"get.metrics",
    "params":{
        "access_token":"string",
        "user_id":"number",
        "report_type": "enum"
    }
}
