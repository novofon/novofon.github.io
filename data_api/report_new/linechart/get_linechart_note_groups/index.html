    Получение списка группированных примечаний

    Получение списка группированных примечаний > Получение списка группированных примечаний
    Метод get.linechart_note_groups
    Описание Получение списка группированных примечаний для отображения в линейном графике отчетов типа "Анализ рекламы"
    Кому доступен Агент, Клиент
    Необходимые компоненты для работы с методом Data API для настройки
    Вес метода 1 балл

    Параметры запроса

    Название Тип Обязательный Допустимые значения Описание
    access_token string да Ключ сессии аутентификации
    user_id number нет Уникальный идентификатор пользователя клиента агента от имени которого делается запрос
    Является обязательным для агента
    Для получения списка пользователей клиента необходимо использовать метод "get.customer_users"
    date_from iso8601 да YYYY-MM-DD hh:mm:ss Дата начала основной выборки(ограничение 366 дней)
    date_till iso8601 да YYYY-MM-DD hh:mm:ss Дата окончания основной выборки
    datetime_dimension enum да
    • hour
    • day
    • week
    • month
    		 Мнемоника измерения. Выбор по какому временному значению буду строиться данные.
    is_show_weekends_notes boolean да Запросить ли список выходных дней для примечаний линейного графика

    Параметры ответа

    Название Тип Допустимые значения Фильтрация Сортировка Ответ по умолчанию Описание
    Массив данных запрашиваемого отчета
    data array нет нет Массив данных запрашиваемого отчета
    datetime_dimension_interval object Временной интегравал примечания - конкретная точка на графике
      from
    		iso8601 YYYY-MM-DD hh:mm:ss Дата начала
      till
    		string YYYY-MM-DD hh:mm:ss Дата окончания
    type enum
    • not_working_days
    • custom
    		 нет нет да Тип примечания
    count number нет нет да Количество примечаний в дне
    is_system boolean нет нет да Является примечание системным или пользовательским
    Список примечаний
    notes array нет нет Список примечаний по дням
    id number Уникальный идентификатор пользовательского примечания
    datetime iso8601 YYYY-MM-DD нет нет Дата примечания
    description string нет нет Содержимое примечания

    JSON структура запроса

    {
    "id": "number",
    "jsonrpc": "2.0",
    "method": "get.linechart_note_groups",
    "params": {
        "access_token": "string",
        "user_id": "number",
        "date_from": "iso8601",
        "date_till": "iso8601",
        "datetime_dimension": "enum",
        "is_show_weekends_notes": "boolean"
     }
}
